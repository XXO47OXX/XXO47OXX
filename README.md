### 🔧 Featured Project

**[spa-reader-mcp](https://github.com/XXO47OXX/spa-reader-mcp)** — MCP server for rendering JavaScript SPAs to LLM-ready Markdown

[![npm version](https://img.shields.io/npm/v/spa-reader-mcp.svg)](https://www.npmjs.com/package/spa-reader-mcp)
[![npm downloads](https://img.shields.io/npm/dm/spa-reader-mcp.svg)](https://www.npmjs.com/package/spa-reader-mcp)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

Traditional web scrapers fail on SPAs. This tool launches Chromium, waits for full render, extracts clean Markdown using Mozilla Readability.

**Key Features:**
- Headless browser rendering with Playwright
- SSRF protection & selector injection prevention
- 100KB output cap with clean truncation
- Singleton browser pattern for efficiency

### 🛠 Tech Stack

`TypeScript` • `Node.js` • `Playwright` • `Chromium` • `Python` • `LLM/AI`
